INTRODUCTION TO HTML5

HTML (Hypertext Markup Language) is the standard language for creating web pages.

HTML5 is the latest version of HTML, which includes new features, elements and improvements on structure together with support for multimedia (audio and video) for modern web development without needind external plugins.

HTML5 is designed to be more efficient, flexible and user-friendly, making it easier for developers to create interactive and dynamic web applications.

TAGS

Tags are semantic elements that provide a better structure for creating and formatting content on a webpage.

Container tags usually come in pairs and have opening and closing tags ie;
<opening tag> and </closing tag>

Examples include:

  1. <h1>for main headings</h1>
  2. <p> for paragraphs </p>
  3. <ul> for unordered lists </ul>
  4. <ol> for ordered lists </ol>
  5. <div> for division of content </div>

Self-closing tags do not have a closing tag and are used for elements that do not require content inside them. ie;
<tagname (attribute) >

Examples include:

  1. <img > for images
  2. <br> for line breaks
  3. <input> for form inputs
  4. <meta> for metadata
  5. <link> for linking external resources

ATTRIBUTES

Attributes provide additional information about an element and are specified within the opening tag. They are used to modify the behavior or appearance of an element.

They are written as name="value" pairs.

Basic syntax of an attribute: <tagname attribute="value" >Content </tagname >

Common attributes include:

  • src: Specifies the source of an image or media file.
  • alt: Provides alternative text for images.
  • href: Specifies the URL for links.
  • class: Assigns a class name to an element for styling purposes.
  • id: Assigns a unique identifier to an element.
Search Engine Optimization (SEO)

Search Engine Optimization (SEO) is the practice of optimizing web pages to improve their visibility in search engine results.

Examples of features in HTML5 that engance SEO include:

  • Semantic elements (like <header>, <footer>, <article>, <section>) .
  • Use of meta tags (like <meta name="description" content="..."/>).
  • Proper use of headings (H1, H2, etc.).
  • Alt attributes for images.
